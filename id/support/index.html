Penagihan Dan Pembelian

  • SiteReportKit tidak memerlukan akun atau login demo untuk aplikasi offline-first saat ini.
  • Aplikasi gratis ini memungkinkan Anda membuat laporan dan pratinjau lokal tanpa batas, dengan 3 ekspor file PDF/CSV yang diberi watermark. SiteReportKit Pro adalah pembelian App Store satu kali untuk ekspor bersih tanpa batas, penghapusan tanda air, dan pencitraan merek khusus.
  • Gunakan Pulihkan dalam dialog Pro untuk memulihkan pembelian non-konsumsi sebelumnya yang dilakukan dengan Apple ID yang sama.
  • Masalah pembelian, penerimaan, pengembalian dana, dan tagihan App Store ditangani oleh Apple. Jika pembelian tidak terbuka setelah Apple mengonfirmasinya, kirimkan email ke dukungan dengan versi aplikasi Anda, model perangkat, versi iOS, dan perkiraan waktu pembelian.
  • Catatan proyek, foto, tanda tangan, PDF, file CSV, dan arsip tetap bersifat lokal kecuali Anda memilih untuk mengekspor atau membagikannya.

Alur Kerja Umum

  • Buat proyek dengan detail klien, situs, inspektur, merek, ukuran kertas, jenis laporan, dan penafian.
  • Tambahkan masalah dengan area, perdagangan, tingkat keparahan, penerima tugas, tanggal jatuh tempo, catatan, dan bukti foto.
  • Gunakan markup foto dan keterangan untuk memperjelas cacat, barang berlubang, barang tersangkut, atau bukti obral.
  • Ekspor laporan lengkap, daftar tindakan kontraktor, laporan obral, register CSV, dan arsip proyek.

File Contoh

Anda dapat meninjau contoh keluaran dari situs produk: contoh laporan tersangkut, daftar tindakan kontraktor, dan menerbitkan CSV daftar.
