Ringkasan

SiteReportKit dirancang sebagai aplikasi pelaporan lapangan yang mengutamakan offline. Data inti proyek, catatan masalah, foto, anotasi, tanda tangan, PDF, file CSV, dan arsip proyek dibuat dan disimpan secara lokal di perangkat Anda kecuali Anda memilih untuk mengekspor atau membagikannya.

Data yang Tidak Kami Kumpulkan

Aplikasi SiteReportKit saat ini tidak menggunakan akun yang dioperasikan pengembang, SDK periklanan, SDK analitik, SDK pelacakan, atau layanan cloud yang dioperasikan pengembang untuk alur kerja inti. Kami tidak menerima catatan proyek, foto, tanda tangan, laporan, pengidentifikasi perangkat, atau analisis penggunaan Anda dari aplikasi.

Data yang Anda Buat di Aplikasi

Anda dapat memasukkan atau mengimpor nama proyek, detail klien dan situs, alamat, detail inspektur, informasi perusahaan, catatan penerbitan, tanggal jatuh tempo, penerima tugas, foto, anotasi, tanda tangan, pengaturan laporan, dan bukti obral. Informasi ini digunakan secara lokal untuk membuat daftar pukulan, snagging, cacat, daftar tindakan, CSV, arsip, dan laporan obral.

Mengekspor Dan Berbagi

Saat Anda mengekspor, menyimpan, mencetak, atau membagikan arsip proyek PDF, CSV, gambar, atau SiteReportKit, data yang dipilih akan meninggalkan aplikasi melalui tujuan yang Anda pilih, seperti File, Mail, Pesan, AirDrop, alur kerja pencetakan, atau aplikasi terinstal lainnya. Tujuan tersebut dikontrol oleh Anda dan mungkin memiliki praktik privasinya sendiri.

Kamera, Foto, Dan File

SiteReportKit hanya meminta akses kamera dan perpustakaan foto agar Anda dapat menangkap atau mengimpor bukti situs, logo perusahaan, dan media laporan terkait. Akses file digunakan untuk impor, ekspor, pencadangan, dan file handoff yang dipilih pengguna.

Anak-anak

SiteReportKit adalah alat pelaporan lapangan profesional dan tidak ditujukan untuk anak-anak. Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak melalui aplikasi.

Layanan Masa Depan

Jika SiteReportKit menambahkan sinkronisasi cloud opsional, akun, langganan, analitik, pelaporan kerusakan, tiket dukungan, atau integrasi di masa mendatang, kebijakan ini dan informasi privasi App Store akan diperbarui sebelum layanan tersebut dirilis.
