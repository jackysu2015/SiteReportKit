ملخص

تم تصميم SiteReportKit ليكون أول تطبيق لإعداد التقارير الميدانية دون الاتصال بالإنترنت. يتم إنشاء بيانات المشروع الأساسية وسجلات الإصدارات والصور والتعليقات التوضيحية والتوقيعات وملفات PDF وملفات CSV وأرشيفات المشروع وتخزينها محليًا على جهازك ما لم تختر تصديرها أو مشاركتها.

البيانات التي لا نجمعها

لا يستخدم تطبيق SiteReportKit الحالي الحسابات التي يديرها المطورون، أو مجموعات SDK للإعلان، أو مجموعات SDK للتحليلات، أو مجموعات SDK للتتبع، أو خدمة سحابية يديرها المطور لسير العمل الأساسي. لا نتلقى سجلات مشروعك أو صورك أو توقيعاتك أو تقاريرك أو معرفات جهازك أو تحليلات الاستخدام من التطبيق.

البيانات التي تقوم بإنشائها في التطبيق

يمكنك إدخال أو استيراد أسماء المشاريع، وتفاصيل العميل والموقع، والعناوين، وتفاصيل المفتش، ومعلومات الشركة، ومذكرات الإصدار، وتواريخ الاستحقاق، والمكلفين، والصور، والشروح، والتوقيعات، وإعدادات التقرير، وأدلة الإغلاق. يتم استخدام هذه المعلومات محليًا لإنشاء تقارير القائمة المثقوبة والتمزق والعيوب وقائمة الإجراءات وملف CSV والأرشيف والإغلاق.

التصدير والمشاركة

عند تصدير أو حفظ أو طباعة أو مشاركة أرشيف مشروع PDF أو CSV أو صورة أو SiteReportKit، فإن البيانات المحددة تترك التطبيق من خلال الوجهة التي تختارها، مثل الملفات أو البريد أو الرسائل أو AirDrop أو سير عمل الطباعة أو أي تطبيق آخر مثبت. يتم التحكم بهذه الوجهات بواسطتك وقد يكون لها ممارسات خصوصية خاصة بها.

الكاميرا والصور والملفات

يطلب SiteReportKit الوصول إلى الكاميرا ومكتبة الصور فقط حتى تتمكن من التقاط أو استيراد أدلة الموقع وشعارات الشركة ووسائط التقارير ذات الصلة. يتم استخدام الوصول إلى الملفات لعمليات الاستيراد والتصدير والنسخ الاحتياطي وملفات التسليم التي يحددها المستخدم.

الأطفال

SiteReportKit هي أداة احترافية لإعداد التقارير الميدانية وليست موجهة للأطفال. نحن لا نجمع معلومات شخصية من الأطفال عن قصد من خلال التطبيق.

الخدمات المستقبلية

إذا أضاف SiteReportKit مزامنة سحابية اختيارية أو حسابات أو اشتراكات أو تحليلات أو تقارير الأعطال أو تذاكر الدعم أو عمليات التكامل في المستقبل، فسيتم تحديث هذه السياسة ومعلومات خصوصية App Store قبل إصدار هذه الخدمات.
